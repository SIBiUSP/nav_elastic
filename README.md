# nav_elastic


## Install

Run: 

mkdir upload

chown -R www-data:www-data upload

create inc/config.php

curl -s http://getcomposer.org/installer | php

php composer.phar install --no-dev

## System requirements

php5-curl

php5-json > 1.3.7
