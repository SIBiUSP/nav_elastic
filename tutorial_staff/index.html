Como funciona a bdpi.usp.br?

  • Registros das Bases de Produção Intelectual (Base 04) e Teses e Dissertações (Base 03) do DEDALUS
  • Sistema de sincronização roda diariamente, durante a madrugada
  • O Registro é convertido de MARC para Schema.org
  • O Registro é enriquecido com fontes externas via APIs (Application Programming Interface)
